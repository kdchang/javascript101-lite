# 從零開始學 JavaScript（JavaScript 程式設計新手村 Lite）
由淺入深系統性學習原生 JavaScript 和程式設計的重要觀念， 讓你不只會用 jQuery 還了解它背後使用的原理，並為未來離開新手村出發挑戰前端進階學習打好堅實基礎知識，培養自學各種前端框架和工具的能力。
